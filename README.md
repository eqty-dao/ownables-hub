# Ownables Hub

> Hub service for Ownables and lockable NFTs

## Installation

```bash
$ npm install
```

Built with [NestJS](https://nestjs.com/).

## Running the app

```bash
# development
$ npm run start
# Swagger => http://localhost:3000/api-docs

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
